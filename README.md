## 🔧 Installation & Setup

1️⃣ **Clone the Repository:**

```bash
git clone https://github.com/abhay/abhay-portfolio.git
cd abhay-portfolio
````

2️⃣ **Install Dependencies:**

```bash
npm install
```

3️⃣ **Run the Development Server:**

```bash
npm run dev
```

4️⃣ **Build for Production:**

```bash
npm run build
```

5️⃣ **Preview the Production Build Locally:**

```bash
npm run preview
```
